SI-CARI

Feb 2021 - Aug 2021

This project is developed for my Bachelor's Degree Thesis. I built the deep learning model with LSTM and GloVe using a fake news dataset from Kaggle in Google Colab. The data preprcoessing is done using the NLTK framework. The machine learning model achieved 99.09% Accuracy and was successfully implemented into a web app using Streamlit and hosted in Heroku.
Tools / Framework Used: Python, TensorFlow, GloVe, Matplotlib, NLTK, Streamlit, GitHub, and Heroku

SWEETECT

May 2021 - July 2021

Sweetect is a web and android base app that could detect sugar and calorie from food Images using Machine Learning Image classification model. The machine learning model was built using TensorFlow and deployed into Google Cloud Run's Good Cloud Platform (GCP) using Docker and Flask. The android was developed in Android Studio and written in Kotlin. I was the team's representative where I was responsible for leading, motivate, manage, make decisions, prepare and make sure we complete the requirement in every document, and oversees while keeping track of the updates from my teammates. For the technical responsibility, I was responsible for training the machine learning model and deploying the machine learning model using Flask, Docker, and using the Google Cloud Run service in GCP. This project is a Capstone Project for Bangkit Academy 2021.
Tools / Framework Used: Python, Kotlin, TensorFlow, Google Colaboratory, Google Cloud Platform, Docker, Flask, HTML, JavaScript, and GitHub.

CEK-LOKER

Aug 2020 - Sept 2020

The Machine Learning model is created to classify fake job vacancies. The model is built using various data preprocessing techniques such as tokenizing, lower-case, stemming, and TF-IDF. The library used for building the machine learning model is scikit-learn. The web application is built using Streamlit and hosted on Heroku. This project was created and submitted as a requirement for the elimination round of Gemastik 13 - Data Mining Competition. This project secured my team (Semicolon) a place in the final round of Gemastik 13 - Data Mining Competition.
Tools / Framework: Python, Scikit-learn, Matplotlib, Streamlit, NLTK, GitHub, and Heroku.
